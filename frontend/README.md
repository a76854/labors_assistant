# 劳动者维权 AI 助手 - 前端工程

本目录是项目的前端工程，基于 `Vite + React 19 + TypeScript + Ant Design 6` 构建。整体设计上采用深浅色双模式适配以及高级现代视觉风格（毛玻璃、立体阴影、CSS 动画），全过程遵循标准化的 Typescript 类型约束。

## 当前技术栈

- React `^19.2.4` & React DOM `^19.2.4`
- Ant Design `^6.3.4`
- React Router DOM `^7.13.2`
- Vite `^8.0.1`
- TypeScript `~5.9.3`

## 🚀 使用方式

### 环境准备

确保您的本地环境已安装 Node.js (推荐 v18+)。在 `frontend/` 目录下执行：

```bash
# 1. 安装核心依赖
npm install

# 2. 启动开发服务器
npm run dev
```

### 常用命令

```bash
npm run lint    # 执行 ESLint 语法与规则检查
npm run build   # 执行 TypeScript 编译并打包压缩至 dist/
npm run preview # 预览本地构建后的产物
```

### 环境变量

项目的 API 地址依赖于环境变量获取，请确认在 `frontend/` 下拥有默认的配置文件 `.env.development`：

```text
VITE_API_BASE_URL=http://localhost:8000
```
（本地部署后端 FastAPI 服务时，默认启动端口为 8000）

## ✅ 已实现功能 (MVP 主链路)

目前前端已完成 MVP（最小可行性产品）版本的所有主链路逻辑，完成了前端界面对后端接口（Routes & Schema）的全面集成与对齐。主要包含：

1. **多轮问答链路**
   - **首页接引**：支持不同案件类型的选择交互，并向后端调用创建对应类型的 Session 初始化。
   - **Chat 流式处理**：拥有完整的发文动作阻塞、助手打字机动画遮罩等待，以及准确拉取后端聊天内容队列（当前回吐固定 Mock 消息）的历史列表展示。
2. **文档生成与渲染反馈**
   - 在对话主视图中支持用户主态调用后端派发“生成诉状”业务指令。
   - 支持生成后基于响应 UUID 路由至带拟真空间特效的 `Result` 展示页，即时展示当前后端该文档在服务器的状态情况（生成中/生成结项/时间戳/物理存储大小）。
3. **状态优雅降级与防御态设计**
   - **界面级自适应**：主页面包含空状态列表提示、加载 Spin 以及规范的业务 Error Message 回馈。
   - **无极双色适配与高级动态光效**：取消所有硬编码颜色，全组件应用纯系统 CSS 变量与设计 Token，实现了能够顺滑无缝切换的 Light/Dark 主题以及自带景深跑马灯的沉浸材质。
   - **诚实阻断反馈**：针对现存的已知限制（如后端文档正文 `content` 暂未组装完成，以及导出接口真实 Buffer 数据未接入），前端专门构建了无干扰界面的骨架屏占位与未开放特性的拦截弹窗，确保了原型体验阶段不会引起用户的断层误解。
